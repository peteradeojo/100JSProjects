Showing Uploaded File Details

Description

This code snippet uses the JavaScript File and FileList APIs to access file information directly from the webpage. Here, we demonstrate a few uses of this technology by choosing the file name, calculating the size in kilobytes and displaying them to the user.

We have also used CSS to display a customized file selection interface to the user. Nothing fancy, just some simple touches. 

		document.getElementById('file').addEventListener('change', function () {
			const file = this.files[0];
			let text = `<p>Name: ${file.name}</p>';
			text += `<p>Size: ${Math.floor(file.size / 1024)} KB</p>`;
			text += `<p>Type: ${file.type}</p>`;
			document.getElementById('file-info').innerHTML = text;
		});
